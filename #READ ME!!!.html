How to Install?

Installing a Loadingscreen is very Easy, in case you used something like an already existing Prefab (e.g. PlumeESX) then you already have an Loadingscreen Resource, you can simply replace the Files.

If you don't already have one defined, then you create a Resource folder (e.g Loadingscreen) and place the Files inside there, now you just add the created Resource to your Config. [or TXAdmin Interface]

How to Edit?

Open the Index.html with your IDE. - I recommend VS-Code - If you know how to Edit SASS then I would recommend editing the SCSS. if you don't, then edit the normal style.css | there are no Comments tho, since my IDE generates the code based on the SCSS: Before editing the SCSS, make sure to have WatchSASS and LiveServer installed.
